Title: 'Dark Style' phpBB3 Smilies
Download Link: https://www.phpbb.com/customise/db/style/dark_style_phpbb3_smilies/
Support: https://www.phpbb.com/customise/db/style/dark_style_phpbb3_smilies/support

This ZIP includes the current phpBB3 smilies which have been altered slightly in order to display correctly on darker styles (removes the white pixels from around the edges).

Compatability: 3.1.0 or above. [*]

Notes: Simply upload to your 'phpBB3/images/smilies' directory, overwriting the existing smilies. You may need to refresh your browser cache for the changes to appear. 

[*] If uploading to a prior version, you'll need to manually resize some of the smilies in the ACP to suit the new dimensions.

Enjoy!
